# python-project-1
💰 Personal Expense Tracker
A feature-rich command-line expense tracker built entirely with Python's standard library — no dependencies needed!

✨ Features
➕ Add expenses with description, amount, date & category
📋 View all expenses in a clean table
📊 Category-wise summary with visual bar chart
🗑️ Delete expenses by ID
📤 Export to CSV for Excel/Sheets
🚀 How to Run
python3 expense_tracker.py
No installation required! Works with Python 3.8+.
